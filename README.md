# jeu_de_taquin

ce projet est fait en binome

le component "tile" et une partie de "grid" est code par monsieur dufourcq

pour jouer ce jeu:

1)cloner le projet, et l'environment a deja maven et npm

2)cd dans le repertoire "backend" et lancer la commend
  mvn spring-boot:run

3)entrer dans le repertoire "backend" et lancer la commend
  npm install

4)une fois installation fait, lancer la commend
  npm start

5)commencer a jouer
